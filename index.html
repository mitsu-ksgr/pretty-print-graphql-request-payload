How to Use

Paste a GraphQL request payload, and pretty print it.

How to copy request payload on Chrome

  1. open your page making the graphql request.
  2. open 'inspect' and, select 'Network' tab.
  3. find the graphql request and click on it.
  4. select 'Headers' > 'Request Payload'.
  5. if you see the 'view source' button, click it.
  6. the payload will appear, and you can copy it!
